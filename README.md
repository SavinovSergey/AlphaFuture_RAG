### Финал хакатона Альфа-Будущее. Трек RAG.
В этом репозитории представлено решение финала хакатона.

В отборочном этапе необходимо было спроектировать retrieval часть pipeline - среди top-5 документов выдать хотя бы один релевантный.
В качестве эмбеддера выбран [ai-forever/FRIDA](https://huggingface.co/ai-forever/FRIDA). BM25 и реранкинг не дали улучшения.

В финале уже попросили разработать generative часть - сгенерировать наиболее полный ответ по сравнению с эталонным и при этом ответ не должен быть очень длинным.
В качестве генератора выбран [t-tech/T-lite-it-1.0](https://huggingface.co/t-tech/T-lite-it-1.0). 

Финалистам был выдан доступ к серверу с NVIDIA L4. Для эффективного инверенса LLM на этом сервере была выбрана библиотека vllm. Ниже приведена команда запуска.
```bash
vllm serve t-tech/T-lite-it-1.0 --h your_host -p 8000 \
  --max-model-len 8192 --gpu-memory-utilization 0.85 \
  --max-num-batched-tokens 4096 --max-num-seqs 16
```
